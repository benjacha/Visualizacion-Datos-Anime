  - **Link a la fuente:** https://www.kaggle.com/datasets/yaminh/anime-ratings-and-popularity-dataset
  - **Descripcion:** The dataset comprises various columns detailing information about anime titles. The "Title" column contains the names of individual anime. "Genres" categorizes each anime into different thematic groups such as action, romance, or fantasy. "Rank" denotes the relative position of each anime based on predetermined criteria, possibly including user ratings or critical acclaim. "Popularity" reflects the level of public interest in each anime, often measured through online metrics like searches or views. The "Score" column indicates the overall rating or quality assessment assigned to each anime, usually derived from user reviews or aggregated scores. "Episodes" specifies the number of installments or chapters within each anime series, while "Episode length" details the duration of each episode in minutes or hours. Lastly, "Release Date" denotes the date when each anime was first aired or made available to the public, providing context about its temporal placement within the anime landscape. Together, these columns offer a comprehensive overview of anime titles, their characteristics, and their reception among audiences.
